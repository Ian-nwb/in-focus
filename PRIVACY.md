# Privacy Policy for in-fcs (In Focus)

**Last Updated: June 2026**

The `in-focus` Chrome Extension is built as a lightweight, user-controlled productivity tool. Your privacy is absolutely paramount, and this policy outlines how data is handled.

### 1. Data Collection & Transmission
- **No Personal Data Collected:** This extension does not collect, store, or monitor any personally identifiable information, browsing history, or user metrics.
- **No Remote Transmission:** Absolutely zero data is transmitted over the internet to the developer or any third-party servers. The extension operates entirely locally within your browser ecosystem.

### 2. Permissions & Local Storage Usage
- **Storage Permission:** Used exclusively to save your custom block list locally on your own machine using Chrome's secure local storage API.
- **declarativeNetRequest & Host Permissions:** Used solely to compare the URLs you manually input into your block list against outgoing navigation requests so the extension can redirect you to the local focus screen.

### 3. Changes to This Policy
Any future updates to this policy will be committed directly to this open-source repository.

### 4. Contact
For any questions regarding this extension, you can open an issue directly on this GitHub repository.
